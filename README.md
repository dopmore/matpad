# BOM:

    - 16x Cherry MX Switches (+1 Just in Case?)
    - 15x DSA Keycaps
    - 16x Diodes (1N4148)
    - 4x M3x5x4 Heatset inserts (After I build the PCB i might remake the case to use screws)
    - 4x M3x16mm SHCS Bolts
    - 1x EC11 Rotary Encoder
    - 1x XIAO RP2040
    - 1x custom PCB
